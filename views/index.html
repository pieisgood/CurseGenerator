How to use

The Curse Generator takes in a certain syntax, so in order to get a proper set of words out, please follow this syntax. Each type of word you want generated must be surrounded by "[" and "]". Also to have more than one word you must add an "+" symbol between word types. All words that don't follow this format will be spit back out by the generator.
Example: [verb] + the + [adjective] + [noun]
This could generate: fuck the fucking fuckers
